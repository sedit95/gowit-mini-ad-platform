# Build Frontend Workflow

This workflow guides the construction of the frontend:
- **Stack**: Vite + React + TypeScript.
- **Views**: Campaign list, campaign creation form, and a detailed campaign view.
- **Features**: Live stats via polling, active/paused/completed status badges, and typed API integration.
- **Design Rule**: Keep UI components simple. Avoid unnecessary UI complexity or heavyweight frameworks beyond the essentials.

*Note: This is workflow guidance only. Do not write implementation code until instructed.*
