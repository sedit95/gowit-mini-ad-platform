# Docker Compose Skill Instructions

When working on Docker orchestration:
- Include services for the Go backend, React frontend, and PostgreSQL database.
- Utilize .env files for environment variable configuration.
- Implement proper health checks (e.g., wait for PostgreSQL to be ready before starting the backend) if it simplifies startup.
- The ultimate goal is a reliable, one-command startup (docker-compose up).

*Note: Do not create the docker-compose.yml file yet.*
