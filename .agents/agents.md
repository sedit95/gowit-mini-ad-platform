# Intended Agent Roles

This document serves as role guidance for the AI agents operating on this project. No external tools or credentials are automatically assigned.

1. **Planning Agent**: Responsible for structuring work, understanding AI_CONTEXT.md, and preventing scope creep.
2. **Backend Go/PostgreSQL Agent**: Handles the implementation of the Go backend, REST API, and PostgreSQL persistence logic.
3. **Frontend React TypeScript Agent**: Handles the Vite + React + TS frontend development.
4. **Race Condition Reviewer**: Scrutinizes the impression deduction logic to guarantee the budget never drops below zero.
5. **Testing and Load Test Agent**: Plans and outlines Go tests and k6 load tests.
6. **Documentation and Workflow Agent**: Maintains honest records of AI interactions, updates documentation, and ensures workflow compliance.
